## API

| 名称     | 描述       | 类型    | 默认值 |
| -------- | ---------- | ------- | ------ |
| type     | 消息的类型 | string  | info   |
| message  | 消息内容   | string  | -      |
| timeout  | 关闭时间   | number  | 3000   |
| closeBtn | 关闭按钮   | boolean | false  |
| offset   | 偏移量     | number  | 20     |
