## API

| Name     | Description     | Type    | Default |
| -------- | --------------- | ------- | ------- |
| type     | message type    | string  | info    |
| message  | message content | string  | -       |
| timeout  | close time      | number  | 3000    |
| closeBtn | close button    | boolean | false   |
| offset   | offset          | number  | 20      |
