## API

| Name     | Description        | Type    | Default |
| -------- | ------------------ | ------- | ------- |
| type     | button type        | string  | primary |
| size     | button size        | string  | md      |
| shape    | button shape       | string  | square  |
| disabled | disable the button | boolean | false   |
