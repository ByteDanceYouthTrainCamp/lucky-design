## API

| Name             | Description | Type   | Default      |
| ---------------- | ----------- | ------ | ------------ |
| right            | 右侧偏移量  | number | 50           |
| bottom           | 底部偏移量  | number | 50           |
| visibilityHeight | 显示的高度  | number | 当前窗口高度 |
