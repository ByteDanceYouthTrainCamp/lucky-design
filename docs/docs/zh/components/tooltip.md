## API

| 名称      | 描述                                        | 类型       | 默认值   |
| --------- | ------------------------------------------- | ---------- | -------- |
| append-to | 指示 Tooltip 的内容将附加在哪一个网页元素上 | CSS 选择器 | -        |
| content   | 显示的内容,也可使用 content 插槽            | string     | -        |
| placement | 出现的位置                                  | string     | 'bottom' |
